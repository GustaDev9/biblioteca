# Library System

## 📌 About

This project simulates a simple library management system.

It was built to practice programming logic, modularization and data manipulation.

## 🛠️ Technologies

- Python
- JSON for data storage

## 🎯 Purpose

The goal of this project is to strengthen backend fundamentals before working with databases and REST APIs.

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/GustaDev9/biblioteca.git
   ```

2. Navigate to the project folder:

   ```bash
   cd biblioteca
   ```

3. Open the file `main.py` and run it in your terminal:

   ```bash
   python main.py
   ```

Make sure Python is installed on your machine before running the project.
